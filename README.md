# Core Foundry - 企业 AI 工程师协同演化基座

> **"AI 时代，每个员工都应该有一套属于自己的‘外挂’，而 Core Foundry 则是我们将这些‘外挂’汇聚成‘军械库’的地方。"**

---

## 🚀 项目背景与愿景

在 AI 浪潮下，能力的鸿沟不再仅仅是知识的沉淀，更是对 AI 工具驾驭能力的差异。
`Core Foundry`（核心铸造厂）的诞生，旨在打破这种不平衡。我们不再是零散的个体在探索 AI，而是通过一个共享的、可进化的“硬核仓库”，将公司内最顶尖的 AI 实践沉淀为 **可复用、可传播、可演化** 的资产。

**我们的目标：**
- **抹平差：** 通过标准化的 Skills 和 Workflows，让每一位员工都能站在巨人的肩膀上，实现 AI 能力的“一键对齐”。
- **提效率：** 不仅仅局限于开发效率，更覆盖行政、运营、财务等全方位的办公生产力。
- **促共建：** 每个人既是使用者，也是贡献者。你的一个精妙提示词，可能就是全公司效率提升的起点。

---

## 🛠 核心板块（持续扩展中）

我们对仓库的定义远不止于“提示词集合”，它是一个多维度的生产力中心：

### 1. ⚡️ AI Skills (原子能力)
*就像是给 AI 安装的专业插件。*
- **结构化 Prompt：** 经过真金火炼的结构化提示词（Role-Task-Constraint 模式）。
- **Python Scripts：** 用于处理枯燥数据的自动化小脚本。
- **MCP Servers：** 扩展 IDE 或 AI Agent 能力的连接器（如文件处理、数据库查询等）。

### 2. 🔄 Workflows (SOP 流程)
*解决“有了工具但不知道怎么配合”的问题。*
- **岗位全流程：** 如“从市场调研到 PRD 生成”的 AI 联动手册。
- **跨部门协作：** 自动化的周报汇总、合规检查、会议总结流水线。

### 3. 🧳 Tools & Environment (军械库)
- **环境预设：** 统一的 VS Code 配置、Cline/Cursor 规则集、本地 LLM 接入指南。
- **私有工具：** 公司内部研发的微内核工具，解决特定业务场景问题。

### 4. 💡 Knowledge & Case (英雄榜)
- **反直觉案例：** 记录那些“意想不到”的 AI 应用妙招。
- **失败记录：** 分享 AI 踩坑经验，避免他人重复试错。

## 🤝 协作共建：每个人都是 AI 工程师

`Core Foundry` 采用 **“开源精神 + 内部闭环”** 的运作模式：

## 📥 快速开始

1. **克隆仓库：**
   ```bash
   git clone git@github.com:your-org/core-foundry.git
   cd core-foundry
   ```
2. **同步技能到本地 IDE (推荐)：**
   执行增强型同步脚本，将仓库内的 Skills 自动安装到你的全能 AI 助手（Antigravity, Cursor, Trae, MarsCode 等）中：
   ```bash
   bash scripts/sync-skills.sh
   ```
   **脚本核心特性：**
   - **自动识别：** 智能扫描 Mac/Linux 环境下已安装的 AI IDE。
   - **双模式选择：** 
     - `开发模式 (Link)`：建立软链接，本地修改实时生效（推荐开发者使用）。
     - `部署模式 (Copy)`：物理复制，生成独立副本。
   - **一键别名：** 选择 `y` 安装 `cf-sync` 命令，未来在任何目录下输入 `cf-sync` 即可秒速同步。
   - **自动清理：** 自动移除本地已过期或不再维护的 Skill。

3. **探索与使用：**
   - 进入 `skills/` 目录查看各个分类下的 `SKILL.md`。
   - 在 IDE 中直接调用已同步的全局技能，体验“开箱即用”的高效。
