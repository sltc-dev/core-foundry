# 技能名称：会议纪要 AI 精炼助手

> 描述：将杂乱的文字转录稿（录音转文字）快速提取为结构化的会议行动项。

## 🎯 适用场景
- 飞书/钉钉会议转录稿清理
- 线下会议录音整理
- 快速同步会议共识

## 🛠 使用方式

### Prompt 模板
```markdown
你是公司最高效的行政助理。请根据以下会议转录文本，生成一份干练的会议纪要。

要求：
1. **摘要**：一句话总结会议核心目的。
2. **议题记录**：记录每一个核心议题及各方主要观点。
3. **TODO List**：提取明确的行动项、责任人及截止时间（如果提及）。
4. **下次跟进**：记录下一次会议的时间或关键节点。

文本内容：
[在此处输入转录内容]
```

## ⚠️ 注意事项 & 技巧
- **噪音剔除**：AI 会自动忽略转录中的“嗯、啊、就是”等口头语。
- **隐私保护**：提交前请注意脱敏敏感的商业或人事数据。
